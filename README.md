# handwerker-webapp
Eine Web-App zur effizienten Verwaltung von Mitarbeitern, Aufträgen und Ressourcen in Unternehmen. Sie bietet Funktionen wie Benutzer- und Rechteverwaltung, Auftrags- und Kalendermanagement, Zeiterfassung, Rechnungsstellung und Integration mit externen Systemen via API. Ideal für eine bessere Organisation und transparente Kommunikation innerhalb des Teams.
